In a Nutshell

Codeception is a new full-stack testing PHP framework. Inspired by BDD, it provides you absolutely new original way for writing acceptance, functional and even unit tests.

Acceptance and Functional Tests

  • Execute with Selenium RC, WebDriver, PhpBrowser, ZombieJS
  • Locate elements by name, CSS, XPath
  • Easily change running backend
  • Cleanup database after each run
  • Integrate with Symfony2, Zend Framework, Kohana, etc.
  • Use with Continious Integration systems like Jenkins or Bamboo

Unit and Integration Tests

  • Two unit test formats
  • Nativley run PHPUnit tests
  • BDD-style scenario-driven tests
  • Smart mocking and stubbing
  • Documentation Generation
  • Generate HTML, XML, TAP, JSON reports
  • Fixtures
  • Database, Doctrine ORM helpers
